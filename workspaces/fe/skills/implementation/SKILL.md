# Implementation Skill

## Workflow
1. Receive task assignment from PM agent
2. Parse requirements and acceptance criteria
3. Update task status to "in_progress"
4. Implement the solution following technical standards
5. Self-review against acceptance criteria
6. Report back to PM with structured response

## Technical Standards
- Use TypeScript with strict mode
- Follow React best practices (functional components, hooks)
- Use Zustand for state management
- Write clean, readable code with meaningful variable names
- Handle edge cases and error states
- Consider accessibility (ARIA labels, keyboard navigation)

## Reporting
Always report back using the structured format:
- Task ID and status
- What was implemented
- Files created or modified
- Any technical notes or concerns
- Known limitations or follow-up items
